# 🎓 Student Result Prediction Using Decision Tree

## 🚀 Project Overview

This project is a **Machine Learning application** that predicts whether a student will **Pass or Fail** based on:

* Study Hours
* Attendance Percentage
* Previous Exam Marks

It uses a **Decision Tree Classifier** from the `scikit-learn` library.

The project also:

* Trains the model
* Calculates accuracy
* Predicts new student results
* Displays a visual Decision Tree

---

## 🛠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

---

## 📊 Features

✅ Creates a student dataset
✅ Splits data into training and testing sets
✅ Trains a Decision Tree model
✅ Predicts result for a new student
✅ Calculates model accuracy
✅ Displays a clean visualization of the decision tree

---

## ▶️ Run
After running, you will see:

* Student dataset printed in terminal
* Prediction for a new student
* Model accuracy score
* Decision tree visualization

---

## 🧠 How It Works

1. Dataset is created manually.
2. Features and target variables are separated.
3. Data is split into training and testing sets.
4. Decision Tree model is trained.
5. Model predicts new student results.
6. Accuracy is calculated.
7. Tree visualization is displayed.
